## 예시4 소수만 찾아내는 함수 만들기

#소수 1과 자기자신만 나누어 떨어지는 1보다 큰 정수

```py
def isPrime(x):
    for i in range(2, x):
        if x%i == 0:
            return False
    return True


a = [12, 13, 7, 9, 19]

for y in a:
    if isPrime(y):
        print(y, end=" ") #13, 7, 19 소수만 출력

```
